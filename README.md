# Text Drop

긴 글을 임시로 업로드하고 삭제할 수 있는 작은 웹사이트입니다. 핸드폰 없이 인증 문구나 긴 메모를 잠깐 옮겨 적어야 할 때 쓰기 좋게 만들었습니다.

## 기능

- 긴 글 붙여넣기에 맞춘 큰 textarea UI
- 모바일 화면 대응
- 업로드 후 공유 링크 생성
- 업로드마다 삭제 토큰 발급
- 삭제 토큰 또는 서버 `ADMIN_KEY`로 삭제
- 1시간/6시간/24시간/3일 자동 삭제 옵션
- JSON 파일 저장소라 별도 DB 없이 실행 가능

## 실행

```bash
npm install
npm start
```

기본 주소: <http://localhost:3000>

## 환경 변수

```bash
PORT=3000                  # 서버 포트
ADMIN_KEY=change-me         # 설정하면 업로드/관리자 삭제에 키 필요
DATA_DIR=./data             # 저장 위치
MAX_TEXT_BYTES=2097152      # 최대 글 크기, 기본 2MB
```

공개 서버에 올릴 때는 반드시 `ADMIN_KEY`를 설정하세요. 이 앱은 임시 메모 전송용이며, 비밀을 장기 보관하는 용도가 아닙니다.
